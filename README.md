# Kaoft
A programming language written with python

# How to use

A simple Hello World:
```
>>> c="Hello World"
>>> c
```

Output:
```
"Hello World"
```

Operators:
```
a=12
b=12
c=a+b
d=12
e=12
f=d-e
h=f*c
```

Output:
```
0
```
