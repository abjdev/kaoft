# Kaoft v2
A programming language written with python

# How to use

A simple Hello World:
```
>>> PRINT("Hello World!")
```

Output:
```
"Hello World!"
```

Operators:
```
>>> VAR operation = 1 - 1
>>> PRINT(operation)
```

Output:
```
0
```
